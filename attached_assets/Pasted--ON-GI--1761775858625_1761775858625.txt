너는 감성적이지만 신뢰감을 주는 랜딩페이지 카피라이터이자 프런트엔드 디자이너야.  
목표는 ‘온기(ON:GI)’라는 새로운 기부 플랫폼의 **사전 오픈 알림 폼(연락처 수집)**을 최대한 많이 받는 것이다.  
HTML이 아니라, **React 컴포넌트 구조로 설계된 랜딩페이지**를 생성해야 한다.  
디자인과 구조는 모바일 웹 중심으로 구성하고, 프런트엔드 배포와 백엔드 연동(폼 제출 API 등)이 용이하게 작성하라.

[서비스 정보]
- 이름: 온기 (ON:GI)
- 컨셉: 3-40대 직장인이 과거의 자신처럼 꿈을 향해 노력하는 20대 청년에게 따뜻한 식사를 선물하며, 공감과 응원을 나누는 연결 기반 기부 플랫폼
- 핵심 메시지: “나와 닮은 청년에게 따뜻한 한 끼를 선물하는 새로운 기부”
- 톤앤매너: 따뜻함, 신뢰감, 감성 + 실천 유도형
- 유저 심리 흐름: (공감) → (이해) → (감동) → (행동, 폼 입력)

[페이지 구성 요청]
1. **Hero 섹션**  
   - 핵심 문장 예시: “나처럼 노력하는 청년에게, 따뜻한 한 끼를 선물하세요.”  
   - 부제: “온기(ON:GI)는 세대 간 따뜻한 연결을 만드는 새로운 기부 플랫폼입니다.”  
   - CTA 버튼: “온기 오픈 알림받기”  
   - 감성 배경 이미지: 따뜻한 식탁, 연결되는 손, 빛의 파동, 미소 등  

2. **Story 섹션 (공감 유발)**  
   - 제목: “왜 ‘온기’가 필요한가요?”  
   - 청년의 현실 서사 예시:  
     “하루 한 끼는 편의점 삼각김밥이 전부인 청년들.  
     누군가의 따뜻한 한 끼가 그들에게 내일을 살아낼 힘이 됩니다.”  
   - 감정 문장: “그때 누군가 나를 도와줬다면… 지금 나는 온기로 보답할 수 있습니다.”

3. **How it works 섹션**  
   - 제목: “한 끼의 온기가 이렇게 전달됩니다.”  
   - 3단계 플로우 컴포넌트  
     ① 한 끼 기부 → ② 청년 식사권 수령 → ③ 감사 메시지 피드백  
   - 시각적 요소: 간단한 일러스트 또는 아이콘, fade-in animation  

4. **Emotional connection 섹션 (후기)**  
   - 제목: “누구나 누군가의 따뜻한 롤모델이 될 수 있습니다.”  
   - 기부자 후기: “예전의 내가 그랬듯, 누군가에게 힘이 될 수 있어 행복했어요.”  
   - 수혜자 후기: “밥 한 끼 덕분에 오늘 하루를 버틸 수 있었어요. 고맙습니다.”  
   - 사회적 증거: “지금까지 1,200명의 직장인이 온기에 함께했어요.”

5. **CTA 섹션 (폼)**  
   - 제목: “당신의 작은 다정이 세상을 바꿉니다.”  
   - 입력 항목: 이름 / 이메일 / 전화번호  
   - 버튼: “온기 시작 소식 받기”  
   - 하단 문구: “스팸 없이 오직 오픈 소식만 전해드려요 :)”  
   - 희소성 카피: “첫 500명에게는 ‘온기 베타 초대장’을 드립니다.”

[디자인 톤 & 스타일 가이드]
- 컬러 팔레트:
  - Warm Cream (#FFF9F3)
  - Soft Lavender (#E9E3FF)
  - Deep Violet (#7C6FF4)
  - Warm Beige (#F6E8DA)
  - Accent Coral (#FFBFAF)
- 전체 톤: 따뜻함 + 차분함 + 희망적인 빛의 느낌
- 폰트: Pretendard, Noto Sans (부드럽고 신뢰감 있는 고딕)
- 여백 충분히, 모바일 퍼스트(375~430px 폭 기준)
- 섹션 간 전환은 fade 또는 subtle slide animation으로 자연스럽게 연결
- CTA 버튼은 각 섹션마다 반복 배치 (sticky 하단 버튼 가능)

[심리적 전환 설계]
- 사회적 증거: “지금까지 1,200명의 직장인이 온기에 함께했어요.”
- 소속감: “당신의 온기가 세대 간 연결의 시작이 됩니다.”
- 희소성: “베타 오픈 알림은 선착순으로 발송됩니다.”
- 감정 몰입: ‘밥 한 끼’라는 현실적 상징 반복 사용

[브랜딩 이미지 가이드]
- Scene 1: 직장인과 청년이 따뜻한 빛 아래 함께 앉은 듯한 실루엣
- Scene 2: 손에서 손으로 전달되는 밥그릇 (OnGi Waves 모티브)
- Scene 3: 발자국이 이어지는 그래픽 (OnGi Footprints)
- Scene 4: 연보라 + 베이지 그라데이션 배경 위에 부드러운 빛 퍼짐 효과

[출력 형태]
- 각 섹션별 카피와 디자인 제안 포함
- 모바일 웹 최적화
